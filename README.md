# DataCheck
特定の階層におけるファイル割合の表示

## 機能
path.txtに記述された階層にあるファイルの割合をメモ帳に出力するアルゴリズムです。

## 動かない場合
・実行できない！<br>
→Golang実行環境がない可能性があります。Golangの実行環境を用意してください。<br>
・Avastでエラーが出る<br>
→Avastのクイックスキャンの関係でプログラムが実行できないことがあります。<br>
例外としてスキャンを外すか、不安ならばプログラムを実行しないことをお勧めします。<br>
